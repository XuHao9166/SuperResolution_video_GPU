# SuperResolution_video_GPU


本项目展示的是：在GPU环境下基于传统的光流算法完成的视频超分辨率重构

依赖环境： opencv3.4.0 版本以上 + cuda8.0 + cudnn 5.1

采用基于稠密光流的4种算法模式 ： farneback, tvl1, brox, pyrlk
